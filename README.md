
# Table Test Task

Небольшой демонстрационный проект — реализация таблицы с пагинацией, поиском, сортировкой и CRUD-поддержкой.

Репозиторий содержит две параллельные идеи реализации одной и той же таблицы:

- Ветка `vue-table` — реализация компонента таблицы с использованием Vue.
- Ветка `ts-table` — реализация той же таблицы без Vue, только на чистом TypeScript.
